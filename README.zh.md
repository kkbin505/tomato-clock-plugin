# 番茄时钟

[English](README.md)

一个简洁的番茄工作法 Obsidian 插件。

## 功能

- 侧边栏面板，显示倒计时、阶段状态和历史记录
- 状态栏显示剩余分钟数，无需打开面板即可感知时间
- 阶段结束时触发系统通知和提示音，切换到其他应用也能收到
- 每次完成工作阶段后，自动在 Markdown 文件（默认 `Tomato Log.md`）追加记录，并打开文件供编辑，支持 `[[双链]]` 关联笔记
- 侧边栏展示本周每日番茄数和历史总计

## 使用方法

点击左侧工具栏的番茄图标打开面板，或通过命令面板执行 `Tomato: Start / Pause`。记录笔记时，请在光标所在行直接输入，不要换行。

## 设置

| 选项 | 默认值 | 说明 |
|---|---|---|
| 工作时长 | 25 分钟 | 每个番茄的工作时间 |
| 短休息 | 5 分钟 | 每个番茄后的短暂休息 |
| 长休息 | 15 分钟 | 完成一组番茄后的长休息 |
| 每组番茄数 | 4 | 触发长休息前的工作次数 |
| 自动开始下一阶段 | 开 | 自动切换工作和休息 |
| 提示音 | 开 | 阶段结束时播放短促提示音 |
| 系统通知 | 开 | 切换到其他应用时仍能收到通知 |
| 日志文件路径 | Tomato Log.md | 记录完成番茄的 Markdown 文件位置 |

## 安装

将 `obsidian-plugin/` 目录下的 `main.js`、`manifest.json` 和 `styles.css` 复制到 vault 的 `.obsidian/plugins/tomato-clock/` 文件夹，然后在 Obsidian 设置中启用插件。
